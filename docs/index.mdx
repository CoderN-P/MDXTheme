# MDXTheme

MDXTheme is a custom MDX theme and CLI tool for building static sites using MDX files. It provides a simple way to create rich documentation with various components and styles.

## Features

- **Customizable**: Easily customize the theme to match your brand.
- **Responsive**: Built with mobile-first design principles.
- **Fast**: Optimized for performance and speed.
- **Markdown Support**: Write content in Markdown and enhance it with MDX components.
